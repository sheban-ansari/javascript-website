# Responsive Interactive Web Application

A fully responsive and interactive web application built using **HTML, CSS, and JavaScript**.  
This project demonstrates core front-end development concepts such as DOM manipulation, event handling, modal implementation, and client-side form validation with a smooth user experience.

---

##  Features

-  Fully responsive design (Mobile, Tablet, Desktop)
-  Interactive dropdown selection
-  Modal popup with multiple close options
   - Close button (×)
   - Background click
   - ESC key support
-  Contact form with client-side validation
-  Success and error feedback messages
-  Auto-hide success message after form submission
-  Smooth and user-friendly interactions

---

##  Technologies Used

- **HTML** – Structure and content
- **CSS** – Styling, responsiveness, media queries
- **JavaScript** – Interactivity, DOM manipulation, event handling
